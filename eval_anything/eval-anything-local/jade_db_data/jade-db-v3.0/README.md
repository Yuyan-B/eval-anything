我们在此发布 JADE 3.0 用于大模型安全微调的高质量中文数据集。

## 数据集说明

我们基于 JADE 1.0 大模型靶向式测试平台生成的高危问题，基于大模型自我反思和修正，生成了高质量的“高危问题-违规回复-安全有用回复”三元组，用于中文大模型的安全对齐。

* **jade_alignment_hard_zh.csv**：大模型安全微调通用高危训练集（基于 JADE 1.0 开源高危测试集生成），包含 150 条“高危问题-违规回复-安全有用回复”的三元组数据；
* **jade_alignment_medium_zh.csv**：大模型安全微调通用进阶训练集（基于 JADE 2.0 开源进阶测试集生成），包含 500 条“高危问题-违规回复-安全有用回复”的三元组数据。

由于相关规定，涉及政治敏感的训练数据目前不包含在可下载的公开训练集中，如需使用，欢迎联系：张谧老师 mi_zhang@fudan.edu.cn

**声明：数据集包含有害违规内容示例，均不代表本团队立场。**
